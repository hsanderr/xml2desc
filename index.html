Sobre

Este programa utiliza o arquivo XML criado pelo Camtasia Studio para gerar uma descrição para o vídeo no YouTube de forma que este seja separado em capítulos.

Como usar

  1. No computador, arraste e solte o arquivo XML sobre a primeira caixa de texto. No smartphone, copie o conteúdo do arquivo XML e cole na primeira caixa de texto.
  2. Clique em Gerar descrição.
  3. No computador, clique em Copiar descrição para copiar o resultado. No smartphone, selecione e copie o conteúdo que apareceu na segunda caixa de texto.

Em breve, um método mais fácil para gerar e copiar a descrição pelo seu smartphone estará disponível.
