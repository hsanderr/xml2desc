Sobre

Este programa utiliza o arquivo XML criado pelo Camtasia Studio para gerar uma descrição para o vídeo no YouTube de forma que este seja separado em capítulos.

Como usar

  1. Colar o texto do XML na caixa de texto da esquerda OU clicar em "Carregar XML" e escolher o arquivo a partir do qual se deseja gerar a descrição.
  2. Clicar em "Gerar descrição".
  3. Copiar a descrição gerada na caixa de texto da direita e colar na descrição de seu vídeo.
